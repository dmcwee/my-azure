# Connect-MyAzure

## SYNOPSIS
Login to an Azure Context

## SYNTAX

```
Connect-MyAzure [[-Environment] <String>]
```

## DESCRIPTION
Login to an Azure Context. 
This function performs the azure login
and then checks if multiple subscriptions exist. 
If multiple 
subscriptions exist the user is prompted to select the desired
subscription.

## EXAMPLES

### EXAMPLE 1
```
Connect-MyAzure
```
